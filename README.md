### 如何使用

在根目录执行
```
node app.js
```
打开浏览器, 转到 <http://localhost:8081> 即可(默认端口)

---

### 主要功能
1. 易懂的操作界面
2. 清晰的权限系统
3. 简单的配置修改


---

### 项目结构
- MSS_3-Back -> 面板后端
- MSS_3-Front -> 面板前端
- MSS_3-Daemon -> 进程守护
